# Zaya Portfolio

Portfolio website built with Next.js.

## Overview

This project is my personal first portfolio site.
It introduces who I am, my background, skills, and projects in a clean and modern format.

## Features

- Responsive portfolio layout
- Hero section with multilingual identity
- Project showcase
- Contact section
- Clean UI with modern styling
- Built with reusable components

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Framer Motion
