## 3. 新プロジェクトの追加レシピ(自分で追加する用)

1. 画像を `public/projects/` に置く
2. `data/projects.ts` の `projectItems` に追加:
   `{ slug, image, tech, demo, github?, status: "Completed", featured: true, kind: "client" | "product" | "sample" }`
   ※ Overland Beyond は `kind: "client"`、タイムアプリは `kind: "product"`
3. `lib/messages/{ja,en,mn}.ts` の `projects.items` に同じ slug で追加:
   `title / description(ベネフィット語で1〜2文) / tags(成果タグ2〜3個) / summary / scope / focus`
   実案件なら `client: "Minuru Motors様"` と `duration: "◯週間"` も
4. 以上。カード・モーダル・LIVEバッジ・実物リンクは自動で付く
   (デモURLがあれば「公開中」バッジと「実物を見る↗」が出る)
